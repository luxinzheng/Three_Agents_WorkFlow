# Installation

## Full install

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/luxinzheng/Five_Agents_WorkFlow/main/install.sh)
```

Use this when you want the full release layout, templates, and docs.

## Safety posture

The installer intentionally does not:
- overwrite Feishu main-channel routing
- auto-enable Telegram
- inject real credentials
- silently replace existing OpenClaw production settings
